## はじめに
tskserver へようこそ。
tskserver は **Minecraft Java Edition** のパブリックサーバです。サーバの雰囲気は[アルバム](/galleries)をみてください。

このサーバ参加ガイドでは tskserver で遊ぶための事前準備の方法や、サーバ接続後の操作方法、コマンドを使った高度な遊び方などを紹介します。
